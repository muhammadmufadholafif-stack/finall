Cloud Infrastructure

Membangun infrastructure AWS dengan aman dan stabil.

IaaS

IaaS merupakan layanan cloud yang menyediakan beberapa platform seperti EC2, dll. Dan pengguna bebas mengatur sesuai kebutuhan.

PaaS

PaaS merupakan layanan cloud yang menyediakan tempat untuk para developer agar bisa mengembangkan dan mendeploy aplikasi tanpa harus mengatur server sendiri.

SaaS

SaaS merupakan layanan cloud yang sudah dideploy atau bisa juga dibilang aplikasi yang sudah siap pakai tanpa harus mengatur server dll. sendiri.
